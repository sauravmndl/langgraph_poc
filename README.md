## How To Use It

```
pip install -r requirments.txt
```

```
python langgraph.py
```

Create .env file and fill up variable **GROQ_API_KEY**. Please check this section for getting api keys https://console.groq.com/keys

```
GROQ_API_KEY="****************"
```
